# Doha Weather CLI

A simple Python CLI tool that fetches live weather data for Doha, Qatar.

## What it does
Fetches real-time temperature, humidity, and wind speed using the 
Open-Meteo API 

No API key required.

## How to run
``` bash
pip install -r requirements.txt
python src/app.py
```

## API used
Open-Meteo — https://open-meteo.com
Free, no sign up, no key needed.
